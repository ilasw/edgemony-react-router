# Edgemony react router

## Dependencies:

installare le dependencies usando il save, per evitare di avere dipenzenze ambigue all'interno del progetto.

```bash

npm install --save [lib]
npm install -D [lib] #dev dependencies

# or

yarn add [lib]
yarn add -D [lib]

```
